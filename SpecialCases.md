# Special git command

Add changes to the last commit and combine both 

```
git commit -amend
```

See all changes
```
git reflog
```

Return to the branch before the accident xD

```
git reset #paste_commit-code --hard
```

# Find a specific word

```
git grep (word)
```

find the line with that word
```
git grep -n (word)
```
count words 
```
git grep -c (word)
```

# Find a word in a commit
```
git log -S (word)
```
